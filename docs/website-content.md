# 0xbuffer — The All-in-One Security Testing Workstation

**See everything. Break everything. Document everything.**

---

## What is 0xbuffer?

0xbuffer is a desktop application that puts a complete security testing toolkit at your fingertips. Whether you are hunting for bugs in a web application, pentesting an API, or mapping out an attack surface, 0xbuffer gives you the visibility and control you need — without needing ten different tools.

---

## Why 0xbuffer?

Modern web security testing is fragmented. You jump between a proxy to watch traffic, a separate tool to intercept and tamper with requests, another to replay them, yet another to fuzz or brute-force, and then a document editor to write your report. 0xbuffer brings all of that into one seamless workspace.

---

## What Can You Do?

### Watch Live Traffic
See every HTTP request and response flowing through your device in real time. Filter by URL, method, status, or custom tags. Build a sitemap of the target as you browse. Drill into any request to inspect headers, cookies, JSON bodies, and more.

### Intercept & Tamper
Pause web traffic mid-flight. Edit the raw HTTP before it reaches the server — or drop it entirely. Test how applications handle malformed input, missing headers, or unexpected parameters.

### Repeat & Craft Requests
Manually compose HTTP requests from scratch and send them to any endpoint. Perfect for testing API behavior, verifying vulnerabilities, or experimenting with different payloads.

### Automate Attacks
Run high-speed brute force and fuzzing campaigns with marked payload positions. Hit login endpoints, enumerate parameters, discover hidden resources — all with real-time progress tracking and response analysis.

### Sniff Raw Packets
Capture raw network traffic from Wi-Fi or localhost interfaces. View hex dumps, reconstruct TCP streams, and parse HTTP from the wire — like Wireshark, but purpose-built for web testing.

### Automate Browser Recon
Let AI drive a browser to crawl target websites on your behalf. It discovers APIs, inventories interactive elements, and logs every action — saving you hours of manual clicking through pages.

### Analyze with AI
Ask an AI assistant to review your discovered assets against OWASP Top 10 frameworks, spot potential weaknesses, or suggest attack vectors. Includes a prompt injection testing tool for evaluating AI system security.

### Build Recon Reports
Document your findings as you work. Create structured reports with markdown sections, API entry catalogs, and custom notes. Export to PDF when you're ready to share.

### Utility Toolkit
Built-in encoder/decoder, hash generators, subdomain enumeration, port scanning, SQL injection testing, GraphQL tools, and more — no need to open a browser tab for quick conversions.
